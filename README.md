مشروع إخفاء المعلومات داخل الصور (Steganography)

فكرة المشروع

هذا المشروع عبارة عن برنامج بلغة Python يقوم بإخفاء الرسائل النصية داخل الصور الرقمية باستخدام تقنية البت الأقل أهمية (LSB).

تعتمد الفكرة على تعديل آخر بت في قيم ألوان الصورة بدون التأثير الملحوظ على شكل الصورة، مما يسمح بإخفاء رسالة سرية داخلها.

مميزات البرنامج

- إخفاء رسالة نصية داخل صورة
- استخراج الرسالة المخفية من الصورة
- استخدام تقنية LSB
- حفظ الصورة الناتجة بصيغة PNG
-

المتطلبات

قبل تشغيل البرنامج يجب تثبيت المكتبات التالية:

pip install pillow numpy

طريقة التشغيل

قم بتشغيل الملف باستخدام الأمر:

python steganography.py

خيارات البرنامج

1. Encode (Hide Message)
2. Decode (Extract Message)

أولاً: إخفاء رسالة داخل صورة

اختر الرقم 1 ثم أدخل مسار الصورة والرسالة.

مثال:

Choice: 1
Enter Image Path: "C:\Users\User\Downloads\image.jpg"
Enter Message: hello

بعد الانتهاء سيتم إنشاء صورة جديدة باسم:

secret_image.png

ثانياً: استخراج الرسالة من الصورة

اختر الرقم 2 ثم أدخل مسار الصورة التي تحتوي على الرسالة.

مثال:

Choice: 2
Enter Image Path: secret_image.png

الناتج:

Hidden Message: hello

ملاحظات مهمة

- يجب استخدام الصورة الناتجة secret_image.png عند استخراج الرسالة.
- لا يُفضل حفظ الصورة بصيغة JPG لأن الضغط قد يؤدي إلى تلف البيانات المخفية.
- يفضل استخدام صيغة PNG لأنها تحافظ على جودة البيانات داخل الصورة.

التقنيات المستخدمة

- Python
- NumPy
- Pillow
